# `set_log`

Customise logs


## Description

Customise the internal logs used by Ambiorix.


## Usage

```r
set_log_info(log)
set_log_success(log)
set_log_error(log)
```


## Arguments

Argument      |Description
------------- |----------------
`log`     |     An object of class `Logger` , see [log::Logger](#log::logger) .
