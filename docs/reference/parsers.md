# `parsers`

Parsers


## Description

Collection of parsers to translate request data.


## Usage

```r
parse_multipart(req, ...)
parse_json(req, ...)
```


## Arguments

Argument      |Description
------------- |----------------
`req`     |     The request object.
`...`     |     Additional arguments passed to the internal parsers.


## Value

Returns the parsed value as a `list` .
